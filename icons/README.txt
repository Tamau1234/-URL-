アイコン生成手順
=================

このフォルダには、Chrome拡張機能用のアイコンが必要です。

方法1: 自動生成ツールを使用（推奨）
--------------------------------
1. generate-icons.html をブラウザで開きます
2. 「すべてのアイコンをダウンロード」ボタンをクリック
3. ダウンロードされた4つのPNGファイルをこのフォルダに保存
4. 完了！

方法2: オンラインツールを使用
---------------------------
以下のサイトで icon.svg を各サイズに変換できます：
- https://svgtopng.com/
- https://convertio.co/ja/svg-png/

必要なサイズ：
- 16x16 → icon16.png
- 32x32 → icon32.png
- 48x48 → icon48.png
- 128x128 → icon128.png

方法3: プレースホルダー画像を使用（一時的）
-----------------------------------------
テスト目的の場合、単色の正方形画像でも機能します。
オンラインツール（https://placeholder.com/）などで
青色（#3b82f6）の画像を各サイズで生成できます。

注意事項
--------
- すべてのアイコンファイルが揃っていない場合、拡張機能の読み込み時に
  警告が表示される場合がありますが、機能には影響しません。
- 正式公開する場合は、必ずすべてのサイズのアイコンを用意してください。
